\# Flutter Music Player UI Skill



This skill is designed to help build a modern and beautiful music player UI

in Flutter applications.



\## What this skill does



\- Provides UI structure for a music player

\- Generates Flutter widget examples

\- Suggests animation and layout patterns

\- Encourages clean and reusable UI components



\## What this skill does NOT do



\- Does not implement backend or streaming logic

\- Does not manage audio files or APIs

\- Focuses only on UI and interaction design



\## Recommended Flutter Packages



\- just\_audio or audioplayers (audio playback)

\- animations (screen transitions)

\- lottie (optional visual effects)

\- riverpod or bloc (state management)



\## Typical Use Cases



\- Music streaming apps

\- Podcast apps

\- Audio book apps

\- AI assistants with media playback
